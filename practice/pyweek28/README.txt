Requirements
------------

	python 2.7+ or 3.5+
	pygame 1.9+

To install the requirements on Ubuntu:

	sudo apt install python python-pygame python-numpy

Running the Game
----------------

Open a terminal / console and "cd" to the game directory and run:

    python run_game.py
